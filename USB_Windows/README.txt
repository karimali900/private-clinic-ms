TO BUILD THE EXECUTABLE:
1. Install Python 3.13+ on your Windows machine
2. Double-click build.bat
3. Wait for build to complete (~5-10 minutes)
4. Output: dist\OMS.exe + data folder
5. Copy dist\* to USB for delivery

TO RUN WITHOUT BUILDING (requires Python):
1. Double-click start.bat
2. Opens at http://localhost:5000/dashboard
