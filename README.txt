====================================================
  SYSTEM REQUIREMENTS
====================================================
• Windows 10/11 (64-bit)
• 4GB RAM minimum, 8GB recommended
• 200MB free disk space
• Screen resolution 1280×720 or higher

====================================================
  INSTALLATION
====================================================
1. Run OMS_Setup_v2.0.0.exe
2. Follow the installation wizard
3. Launch OMS from Start Menu or Desktop shortcut

====================================================
  QUICK START (Portable USB)
====================================================
1. Copy the entire OMS folder to a USB drive
2. Double-click start.bat
3. The app opens at http://localhost:5000/dashboard
